# Dashboard app
### Це навчальна браузерна программа з такими функціями як:
* **TODO** – місце де ви можете зберігати для себе плани на день/тиждень або навіть місяць
* **NOTES** – ваш записничок де ви можете писати що завгодно(ліміт 2500 символів)
* **CALCULATOR** – це просто калькулятор(поки що немає дужок)
* **SAVE URLS** – місце де ви можете зберігати любі посилання(**URL**) та давати їм назву для швидкого розуміння
* **THEME SWITCHER** – перемикач теми з світла => темна

## На сайті викоистовується фейковий сервер для зберігання данних
### > **Дуже рекомендується не спамити запитами та після кліку дочекатись відповідного повідомлення**
