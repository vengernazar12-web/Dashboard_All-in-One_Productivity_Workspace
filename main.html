What is Dashboard

      

        Dashboard is a system for managing data and services.
        It allows you to work with todos, notes,
        codes, urls, and built-in services.
      

    


    

      
Command Format

      

        Commands use the structure:
        action type, field: value
      

      

        Values are separated by commas.
        Multiplier is supported: value*n
      

    


    

      
Actions

      

        
ADD / CREATEAdd items

        
DELETE / REMOVEDelete items

        
SEARCH / FINDSearch items

        
LIST / SHOWShow list

        
INFO / READ / CHECKDetailed information

        
SAVESave type

        
CLEARClear type

        
OPENOpen section

        
CALCEvaluate expressions

      

    


    

      
Data Types

      

        
todosTasks

        
notesNotes

        
codesCode snippets

        
urlsLinks

      

    


    

      
Command Examples

      

        
Addadd todos, names: Task tags: work

        
Searchsearch codes, keys: auth

        
Listlist notes

        
Infoinfo todos, names: Task

        
Openopen settings

        
Calculatecalc 12+4*2

      

    


    

      
How AI Works in Dashboard

      

        The system first tries to process the request using the
        standard command parser.
      

      

        AI is triggered when:
        
— the text does not match command format
        
— the request is written in natural language
        
— the intent is ambiguous
        
— the parser cannot determine the action
      

      

        When triggered, AI assistant analyzes the request and:
        
— or responds with text
        
— or performs an action via the system
      

      

        This acts as a fallback mechanism that enables
        flexible free-form interaction.
